# Резервная копия проекта (30.06.2026)

Эта папка содержит стабильную версию основных файлов сайта до начала ваших экспериментов.

## Что внутри
* `App.tsx` — Главный файл приложения (без всплывающих уведомлений «Живой заказ с сайта»).
* `components/OrderForm.tsx` — Форма заказа с автоматической заменой цифр `8` или `7` на код `+7`.
* `components/LeadMagnetModal.tsx` — Окно лид-магнита с аналогичной автозаменой ввода телефона.

## Как восстановить
Если в процессе экспериментов что-то пойдет не так, вы можете просто заменить текущие файлы в папке `/src` содержимым файлов из этой папки.

## Полный бэкап проекта на компьютер
Вы также можете в любой момент скачать полную резервную копию проекта (со всеми стилями, шрифтами и конфигурациями) в виде **ZIP-архива** или экспортировать проект на **GitHub**. 
Для этого нажмите на **иконку шестерёнки (Settings)** в верхнем правом углу интерфейса AI Studio и выберите соответствующий пункт экспорта.
